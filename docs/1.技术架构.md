# AI 笔记本项目

本项目是一个类似 NotebookLM 与 Open-Notebook 的 AI 笔记本应用，采用前后端分离架构：

- **前端**：Vue 3.6 + Rolldown-Vite，运行时 Bun
  - 状态管理：Pinia
  - 路由：Vue Router
  - 组件库：Naive UI
  - 图标库：Iconify + unocss 图标组件
  - 样式：unocss 原子化 CSS
  - 网络请求：Alova
- **后端**：ElysiaJS，运行时 Bun  
- **数据库**：SurrealDB  

## 快速开始

### 环境要求
- Bun ≥ 1.2

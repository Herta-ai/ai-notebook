# AI 笔记本项目

本项目是一个类似 NotebookLM 与 Open-Notebook 的 AI 笔记本应用，采用前后端分离架构：

- **前端**：Vue 3.6全家桶（Vue Router、Pinia） + Rolldown-Vite，运行时 Bun
  - 组件库：Naive UI
  - 图标库：Iconify + unocss 图标组件
  - 样式：unocss 原子化 CSS
  - 网络请求：Alova
- **后端**：ElysiaJS，运行时 Bun
- **数据库**：SurrealDB
- **桌面端**：Tauri，界面使用前端代码，rust侧使用嵌入式SurrealDB提供后端服务

## 快速开始

### 环境要求
- Bun ≥ 1.2
